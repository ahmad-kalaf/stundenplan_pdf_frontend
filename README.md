# stundenplan_pdf_frontend

Dieses Projekt implementiert das Frontend für eine App zum
Erstellen von Stundenplänen.

Anforderungen:

1. Als Nutzer kann ich ein Modul speichern.
2. Als Nutzer kann ich all meine gespeicherten Module ansehen.
3. Als Nutzer kann ich ein gespeichertes Modul bearbeiten und löschen
4. Als Nutzer kann ich einen Stundenplan mit meinen gespeicherten Modulen generieren lassen.
5. Als Nutzer kann ich meinen erstellten Stundenplan ansehen und löschen
6. Als Nutzer kann ich meinen erstellten Stundenplan als pdf-Datei herunterladen
7. Als Nutzer kann ich ein gespeichertes Modul mit allen Details als Text kopieren
